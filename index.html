Desarrollo de Página Web

Si quieres empezar en el desarrollo aqui tienes la información

1.- ¿Que es?

Es un documento digital multimedia (contiene texto, audios, videos) estos son adaptados para la World Wide Web (www).
Una página es un sitio hecho en internet, estas estan hechas para dar un servicio o informacion sobre un tema de interes.Estas paginas son de dos tipos los cuales son: estáticas o planas, dinámicas.

  • Páginas Estaticas: estas paginas estan hechas para brindar informacion a traves de texto, audio o video de una forma mas visual. En estas paginas el usuario no tiene algun tipo de interacción mas que lo visual
  • Páginas Dinamicas: lo que diferencia una pagina web dinamica con una estatica, es que su interacción es mayor siendo que puede guardar información, enviar informacion, o recibir informacion en nuestra pagina web

2.- ¿Diferencia entre diseño y desarrollo web?

Las diferencias que hay se debe a los dos ámbitos que se traba para realizar una página web, estas son el Frontend y el Backend. El frontend se dedica al diseño y personalización de una página web mientra que el backend se dedica al funcionamiento interno dentro de la página web siendo estas: Las bases de datos, optimización o alguna función fuera del cliente

3.- ¿Cuál es la estructura de una página web?

Se estructura de la siguiente manera:

  • Encabezado: normalmente esta en la parte superior de una página web, este puede contener el logo de tu marca o empresa, un menu de navegación o algo de intéres para que el usuario pueda interactuar con el
  • Cuerpo: aquí se encuentra el contenido principal que queremos mostrar, puede ser un producto, una guía, o algo relevante a mostrar.
  • Pie de pagina: En este apartado se encuentra toda la información sobre contactos, de donde se estrago la información, menciones a alguna corporación o alguna marca registrada

4.- Lenguajes usados para el desarrollo web

  • HTML
  • CSS
  • JAVASCRIPT
  • REACT
  • NODE JS
